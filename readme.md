# 桌面代做任务清单,MyTodoList

### 功能迭代

1. 显示一个可以增加，删除，编辑，开始的todolist.

   里面的任务显示在一个可以滚动的空间里面，所以可以无限地添加

2. 增加文件存储功能，退出程序后再进入时显示的仍然是是上一次自己输入的任务

### 模块划分

### 遇到的问题和解决的技术方案

1. 在有限的区域内显示更多的任务

   使用滚动面板JScrollPanel。任务作为控件添加到JPanel上，再通过一个以该JPanel为参数构造的JScrollPane实例对该JPanel进行管理，使之成为滚动面板
